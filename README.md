# Meilian Tỷ Giá V2 (Theme xanh, bố cục giống mẫu)

## Chạy local
```bash
npm install
npm run dev
```

## Admin
- `/admin`
- Set env trên Vercel: `ADMIN_PASSWORD`

## Lịch sử biểu đồ
- Mỗi lần bấm **Lưu** trong `/admin` sẽ tự thêm 1 điểm lịch sử (tối đa 10 điểm gần nhất)
- Copy JSON -> dán vào `data/config.json` -> commit GitHub để cập nhật lên production
